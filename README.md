Лабораторная работа №9 ( Куда я поеду)

Адаптивный одностраничный сайт-портфолио о путешествии в России, созданный по макету Figma.

Организация командной работы:
Мы работали по описанным в методичке правилам:

. Создана организация GitHub: публичная, с командой из двух разработчиков, оба участника имеют права Maintainer.

. Технологии: В настоящем проекте были использованы принципы адаптивной верстки сайтов с использованием медиазапросов, относительных размеров блоков, grid и flex контейнеров для контента, что позволило сделать сайт удобным для просмотра на устройствах, имеющих разрешение экарна отличное от десктопа.

. Настроена Project Board
Колонки: Todo, In Progress, Done,
Каждая задача была вручную перенесена в нужный статус по мере выполнения.

. Защита веток: main и develop защищены, запрещён прямой push, включены Pull Request review.

. Адаптивность
Сайт проверен на разрешениях:
1280+ px — большой экран
1024 px — десктоп
768 px — планшет
320–480 px — смартфон

Используются: резиновые контейнеры, гибкие сетки, корректные брейкпоинты.

. Проверка качества

Перед публикацией мы проверили:
соответствие макету через PerfectPixel
корректность структуры БЭМ
отсутствие горизонтального скролла
читаемость шрифтов
корректную работу картинок
адаптивность блоков

. Публикация

GitHub Pages включён через:
Settings → Pages → Deploy from branch → my main
Сайт грузится из корневой папки проекта.

Разработчик 1: Гасинец Дмитрий — занимался структурой проекта, верстка header, places, cards, проверил по чек-листу, опубликовал на GitHub Pages.

Разработчик 2: Орлов Никита — занимался версткой intro, footer, people, form, адаптивной версткой, подключением шрифтов и всех стилей к проекту.
